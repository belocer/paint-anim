# gulp4 / Animation paint

### Создать зависимости
`npm i`

### Собрать сборку   
`gulp build`


### Запустить для разработки  
`gulp`


### Удалить все изображения   
`gulp cleanimg`


### Генерировать сетку SmartGrid Дмитрия Лаврика   
`gulp grid`


### Генерировать Критические и Асинхронные стили
`gulp criticalStart`
